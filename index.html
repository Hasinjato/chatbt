Automatisation de publication Facebook

Configuration requise

Avant d'utiliser cet outil, assurez-vous d'avoir:

  • Un jeton d'accès avec les permissions pages_manage_posts et pages_read_engagement
  • L'ID de votre page Facebook
  • Une URL vers un fichier JSON valide contenant vos publications

Automatisation quotidienne

Pour une automatisation quotidienne, vous devrez configurer un serveur ou utiliser un service comme:

  • Google Apps Script (gratuit)
  • Un serveur Node.js avec agenda/cron
  • Services cloud comme AWS Lambda, Google Cloud Functions, etc.
  • Services d'automatisation comme Zapier ou IFTTT (version payante)

Le code JavaScript fourni ci-dessous peut être adapté pour ces services.
